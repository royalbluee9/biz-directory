## 🎯 What Can You Do?

### As a Regular User 👤
1. ✅ **Browse Businesses** - Search 2 pre-loaded sample businesses
2. ✅ **Add Your Business** - Click the + button, fill the form
3. ✅ **Select Country Code** - Choose from 30+ countries with flags
4. ✅ **Request Verification** - Get a verified badge (blue checkmark)
5. ✅ **Chat with Businesses** - Send messages and get auto-replies
6. ✅ **View Messages** - See all conversations in one place

### As an Admin 👑
Everything above, PLUS:
1. ✅ **Admin Dashboard** - Click ⚙️ icon in header
2. ✅ **View All Users** - See who's registered
3. ✅ **Approve Verifications** - Review and approve business verification requests
4. ✅ **System Analytics** - Monitor users, businesses, and verifications

## 📱 Try These Features First

### 1️⃣ Login as Admin
```
Email: admin@bizdirectory.com
```
- Notice the **purple badge** next to your name
- See the **⚙️ settings icon** in the header

### 2️⃣ Open Admin Dashboard
- Click the **⚙️ icon**
- Explore 3 tabs:
  - **Overview** - See system stats
  - **Users** - View all 3 demo users
  - **Verifications** - No pending requests yet

### 3️⃣ Logout and Login as User
```
Email: john@example.com
```
- Notice the **blue badge** (regular user)
- No ⚙️ settings icon (admin only)

### 4️⃣ Add a Business
- Click the **+ button** (bottom-right)
- Fill in details:
  - Name: "Joe's Pizza"
  - Category: Restaurant
  - Description: "Best pizza in town!"
  - **Country:** 🇺🇸 United States (+1)
  - **Phone:** 555-7890
  - Email: joe@pizza.com
  - Address: 789 Main St
- Submit!

### 5️⃣ Request Verification
- Open your business (Joe's Pizza)
- Click **"Get Verified"** button (blue gradient)
- Status changes to **"⏳ Verification Pending"**

### 6️⃣ Approve Verification (as Admin)
- Logout, login as **admin@bizdirectory.com**
- Click **⚙️ icon** → **Verifications tab**
- See your pending request
- Click **"✅ Approve"**
- Business is now verified!

### 7️⃣ See the Verified Badge
- Go back to business list
- See **blue ✓ checkmark** on "Joe's Pizza"
- Open business details to see full verified badge

### 8️⃣ Chat with a Business
- Tap **"Chat"** on any business
- Type: "What are your hours?"
- Send message
- Wait 1 second for auto-reply
- Go back and check **Messages icon** for unread count

## 🎨 Visual Guide

### Header Icons
- **← Back** - Return to business list
- **💬 Messages** - View all conversations (red badge = unread count)
- **⚙️ Settings** - Admin dashboard (admins only)
- **🚪 Logout** - End session

### Badges & Indicators
- **✓** (blue) = Verified business
- **⏳** (orange) = Verification pending
- **❌** (red) = Verification rejected
- **Purple circle** = Admin user
- **Blue circle** = Regular user
- **Red number** = Unread message count

### Buttons
- **+ (bottom-right)** = Add new business
- **Get Verified** (blue gradient) = Request verification
- **Approve/Reject** (green/red) = Admin verification actions
- **Chat** (indigo) = Start conversation

## 🐛 Troubleshooting

### "I can't see Admin Dashboard"
→ Login with **admin@bizdirectory.com** (must be admin role)

### "Get Verified button not showing"
→ You must be the business owner (created it with your account)

### "No messages showing"
→ Send a message first, wait 1 second for auto-reply

### "Can't delete business"
→ You can only delete businesses you created
