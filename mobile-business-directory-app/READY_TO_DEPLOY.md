## 🎯 Deploy Now (Choose One Method)

### Method 1: Automated Script ⭐ RECOMMENDED

**For Mac/Linux:**
```bash
chmod +x deploy.sh
./deploy.sh
```

**For Windows:**
```bash
deploy.bat
```

The script will:
1. ✓ Initialize Git repository
2. ✓ Prompt for your GitHub repository URL
3. ✓ Add and commit all files
4. ✓ Push to GitHub
5. ✓ Show you next steps

### Method 2: Manual Commands

1. **Create a repository on GitHub** at https://github.com/new
2. **Copy the repository URL** (e.g., `https://github.com/username/biz-directory.git`)
3. **Run these commands:**

```bash
git init
git add .
git commit -m "Initial commit: BizDirectory mobile app with chat"
git remote add origin YOUR_REPOSITORY_URL_HERE
git push -u origin main
```

4. **Enable GitHub Pages:**
   - Go to repository Settings → Pages
   - Select "GitHub Actions" as source

5. **Wait 2 minutes** for deployment

6. **Access your app** at:
   ```
   https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/
   ```
